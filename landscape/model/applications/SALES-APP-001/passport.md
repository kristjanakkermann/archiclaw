# Salesforce CRM (SALES-APP-001)

Customer relationship management platform handling sales pipeline, opportunity management, and customer data. Primary interface for the sales team.

## Key Facts

- **Salesforce Enterprise Edition** (SaaS, tri-annual releases)
- Master for customer and opportunity data in the sales domain
- REST API integration with SAP ERP for order-to-cash process
- Extensive custom Apex triggers and Lightning components

## Criticality

**Tier 1** system for the Sales domain. Sales team productivity directly tied to system availability.
