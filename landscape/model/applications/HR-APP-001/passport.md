# Workday HR (HR-APP-001)

Cloud-based human capital management platform handling core HR, payroll, talent management, and workforce analytics. Master system for employee data across the enterprise.

## Key Facts

- **Workday HCM** (SaaS, bi-annual updates)
- Master source for employee records, org hierarchy, and compensation
- Bidirectional RFC integration with SAP ERP for payroll posting and cost allocation
- REST API feed to recruitment platform

## Criticality

**Tier 1** system. Downtime impacts payroll processing, employee self-service, and compliance reporting. GDPR applies to all employee personal data.
